# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-01-19

### Added
- Added SolarOrange theme variant
- Added new semantic tokens for info states

### Changed
- Improved dark mode color contrasts
- Updated build process to support multiple themes

## [1.1.0] - 2026-01-10

### Added
- Added bridge compatibility layer for shadcn components
- New export paths for individual themes

### Changed
- Reorganized theme structure (primitives → semantic → bridge)

## [1.0.0] - 2026-01-05

### Added
- Initial release
- iGreen theme with full token system
- Dark mode support via `.dark` class
- OKLCH color space for better perceptual uniformity
